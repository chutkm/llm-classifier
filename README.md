# llm-classifier

Проект **llm-classifier** — это инструмент для классификации текстов с помощью больших языковых моделей (LLM). 

---

## Описание

`llm-classifier` предоставляет простой способ для классификации текстовых данных, используя возможности современных LLM. Он позволяет:

- Легко создавать классификаторы на основе примеров и/или описаний классов.
- Использовать предобученные LLM,используя модели с Hugging Face.

---


## Результаты:

## Структура
```
llm-classifier/
├── config/                # Конфигурации для задания : модели,промта,групп классификации,пути для сохранения.
├── data/                  # Входящие данных
├── outputs/               # Результат работы
├── scripts/               # точка запуска
├── src/                   
│   ├── classifier.py      # Логика классификации
│   ├── llm_adapter.py     #  LLM-клиент
│   └── utils.py           # Вспомогательные модули
├── requirements.txt       # Зависимости Python
├── README.md              # Документация проекта
└── .gitignore
```


## Требования

Проект написан на Python (3.8+). Основные зависимости описаны в `requirements.txt`.  

---

## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/chutkm/llm-classifier.git
   cd llm-classifier
   ```
2.Установите зависимости:
```
pip install -r requirements.txt
```
3.Настройте переменные окружения для доступа к LLM API:
```
export HF_TOKEN=""
```
